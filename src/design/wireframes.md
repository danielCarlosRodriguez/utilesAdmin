**Notes:**
- Mobile header: Simplified, hamburger menu
- Navigation: Bottom nav bar optional (Home, Search, Cart, Account)
- Product cards: 2-column grid or single column (toggle)
- Filters: Bottom sheet (slide up from bottom)
- Cart: Sticky checkout bar at bottom
- Checkout: Single-column layout, collapsible order summary
- Touch targets: Minimum 44x44px for all interactive elements
- Forms: Full-width inputs, larger tap targets
- Images: Swipeable galleries on product detail

---

## Responsive Breakpoints Summary

| Breakpoint | Width | Layout Changes |
|------------|-------|----------------|
| **Mobile** | < 768px | Single column, hamburger menu, bottom sheet filters, swipeable galleries |
| **Tablet** | 768-1024px | 2-3 columns, visible nav, sidebar filters (collapsible) |
| **Desktop** | 1024-1280px | 3-4 columns, full nav, permanent sidebar filters |
| **Large** | > 1280px | 4-5 columns, wider containers, more whitespace |

---

## Accessibility Notes

### Focus Management
- All interactive elements keyboard accessible
- Visible focus indicators (outline)
- Logical tab order
- Skip links for navigation

### Screen Readers
- Semantic HTML (header, nav, main, footer)
- ARIA labels for icons and buttons
- ARIA live regions for dynamic updates
- Alt text for all images

### Color & Contrast
- WCAG AA compliance (4.5:1 for text)
- Don't rely on color alone (use icons + text)
- High contrast mode support

### Touch & Interaction
- Minimum 44x44px touch targets
- Spacing between interactive elements (8px+)
- No hover-only functionality (mobile)
- Swipe gestures optional (provide alternatives)
